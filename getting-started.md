# Getting Started



## Download's

Here you can download Madium.

* [https://getmadium.online/](https://getmadium.online/)

{% hint style="warning" %}
This is the official Madium website—our only trusted source. Be cautious of any other sites claiming to be Madium, as they’re likely scams trying to trick you into downloading fake software. Stay safe and stick with us!
{% endhint %}





## Setup

After visiting one of the download links you should be inside an Madium Server.

* If you are in Madium Server visit the [`#downloads`](https://discord.com/channels/1483453559692595252/1483498936680317049)  channel and complete the work ink from the latest download.

> After completing Work Ink you should be taken to a GoFile page containing `Madium-Launcher`. Which you should download.



## Installation

Now that you've downloaded the Madium bootstrapper you can launch it.

![](<.gitbook/assets/madium bootystrapper.png>)

_After it opens it should download all required files and launch Madium UI for you._



## Usage

Now that you have the Madium interface you can start by attaching it to Roblox as provided bellow

<figure><img src=".gitbook/assets/madium booty ui.png" alt=""><figcaption></figcaption></figure>

_Simply launch your Roblox and click Attach, after it attaches the button would turn green and say Attached as shown in image bellow._



<figure><img src=".gitbook/assets/madium booty ui attached.png" alt=""><figcaption></figcaption></figure>

_At this step u can normally execute scripts and they should work. If your Roblox crashes check out the_ [support.md](support.md "mention")  page.&#x20;
