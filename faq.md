# FAQ





## Is it completely free?

{% hint style="success" %}
Yes, Madium is and will always be free. We're not like Volcano—we’re committed to being honest and trustworthy with our users. Our goal is to put our best effort into making Madium the top keyless and free Roblox executor, and we’re excited to continue improving it for everyone!
{% endhint %}

## Is there Auto Update?

{% hint style="success" %}
Madium <mark style="color:$danger;">doesn't</mark> update automatically to ensure your safety. When you're ready to update Madium, simply run its bootstrapper file each time—no need to redo your Work.Ink. It's a simple process designed to keep things smooth and secure for you!
{% endhint %}

## Is this undetected?

{% hint style="success" %}
Madium does have a Client Modification Bypass, but there's a chance it could be detected in future ban waves. This means that while Madium is unlikely to get you banned during regular use, we still recommend using alternate accounts just to stay on the safe side.
{% endhint %}

## Is it keyless?

{% hint style="success" icon="octagon-check" %}
Madium is currently completely keyless. The way Madium generates revenue is through advertisements displayed inside Work.Ink when users download the executor.
{% endhint %}

## Have been crashing issues fixed?

{% hint style="success" %}
To the best of our knowledge, all injection-related issues have been resolved. If you still experience crashes, give the fixes inside [support.md](support.md "mention") a try—they might just solve the problem for you.
{% endhint %}

## Is there Auto Execute?

{% hint style="success" %}
Absolutely! Madium does have Auto Execute. All you need to do is create your .lua file inside `%appdata%\Madium\AutoExecute`, and it will automatically run whenever you attach. It's a simple and handy feature to make your experience even smoother!
{% endhint %}

## How to add configs?

{% hint style="info" %}
To include your configs in your favorite scripts, simply add them to your script folders within the Workspace directory. You can find this inside the `%appdata%\Madium\Workspace`  folder. Have fun scripting!
{% endhint %}









## Is UE compatible with this executor?

{% hint style="success" %}
Madium generally works well with Unnamed Enchantments, so you shouldn’t encounter any problems unless the developers update their script in an incompatible way.
{% endhint %}

## Is KiciaHook compatible with this executor?

{% hint style="success" %}
Madium is generally very compatible with Kicia Hook, so you probably won’t encounter any problems unless the developers make an unexpected update to their script.
{% endhint %}







## What is Raknet?

{% hint style="info" %}
[raknet.md](functions/raknet.md "mention")  Is a cross platform, open source, C++ networking engine for game programmers.

Roblox has its own implementation of it which can be abused with packet manipulation allowing you to become basically invisible to other players,  but since this function tampers with Roblox packets its highly un-safe and could get you easily banned if used without caution.
{% endhint %}

## What do i do if i get banned by Roblox Ban API?

{% hint style="info" %}
The best way to resolve this issue is by using Altbot, which you can find at [https://altbot.win/](https://altbot.win/). Once you download it, you'll need to get a key—don't worry, the key system is straightforward and doesn’t require any extra downloads. Just click on a task, wait a few seconds, then go back to the tab. When you receive your key, activate Altbot, and then click on the Spoofer tab inside the app. You'll be prompted to select your Wi-Fi adapter—simply choose it and click Spoof. You can also use the alternative accounts provided by Altbot if you launch a game from the Games tab.
{% endhint %}







## Is this compatible with UWP?

{% hint style="danger" icon="x" %}
Madium isn't compatible with the Microsoft Store version of Roblox, as the web version is quite different from the UWP release.
{% endhint %}

## Is this compatible with MacOS?

{% hint style="warning" icon="circle-info" %}
Madium isn't compatible with MacOS directly, but don't worry—you can easily set up Windows on your Mac using Bootcamp. Once you've installed Windows, you'll be able to enjoy Madium without any trouble!
{% endhint %}

## Is this compatible with Android?

{% hint style="danger" icon="x" %}
Madium isn't compatible with Android, but we're always working to improve your user experience!
{% endhint %}

## Is this compatible with Linux?

{% hint style="warning" %}
Madium isn't compatible with Linux directly, but you can try using Winboat to install Windows inside a Docker container. This way, you'll be able to run Madium. Just keep in mind, I’m not sure if Roblox will work perfectly in this setup, but it’s a good workaround to explore!
{% endhint %}
