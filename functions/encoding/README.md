# Encoding



## Overview

These functions allow you to:

* Convert data to and from Base64
* Compress and decompress data using LZ4

## Available Functions

| Function                            | Description             |
| ----------------------------------- | ----------------------- |
| [`base64decode`](base64decode.md)   | Decode a Base64 string  |
| [`base64encode`](base64encode.md)   | Encode data to Base64   |
| [`lz4compress`](lz4compress.md)     | Compress data using LZ4 |
| [`lz4decompress`](lz4decompress.md) | Decompress LZ4 data     |
